# 数据结构与算法相关概念

## 级数
1. 算数级数：与末阶平方同阶  
T(n) = 1 + 2 + 3 + 4 + n = O(n^2)  
2. 幂方级数：比幂次高出一阶  
T4(n) = 1^4 + 2^4 + 3^4 + n^4 = O(n^5)  
3. 几何级数：与末项同阶  
Ta(n) = a^0 + a^1 + a^2 + a^n = O(a^n)   
T2(n) = 2^0 + 2^1 + 2^2 + 2^n = O(2^n)  
4. 收敛级数：O(1)  
5. 调和级数：   
h(n) = 1/1 + 1/2 + 1/3 + 1/n = O(logn)  
6. 对数级数：  
log1 + log2 + log3 + logn = log(n!) = O(nlogn)  

## 有序  
任意一对相邻的元素都是顺序的  
## 无序  
总有一对相邻的元素时逆序的
